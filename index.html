LightningApp

The LightningApp project is a mobile application developed for devices running on the Android OS. It is designed for news viewing, allowing users to select their preferred news categories. Articles from the chosen categories are sorted into a separate tab. The application is written in Java, utilizing the Retrofit library, the Firebase platform, and so on.
